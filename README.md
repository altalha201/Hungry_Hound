<h1 align="center">Hungry Hound</h1>
<p align="center">A food ordering application.</p>

##### Customer Side
- See restaurant's menu
- Order food 
- Get notify when order accepted/rejected

##### Restaurant Side
- Make menu
- Get notify when a order is made
- Accept/Reject Order
